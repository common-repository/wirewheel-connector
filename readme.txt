== Description ==

This plugin integrates with Wirewheel instance and render the forms easily within Wordpress site.

== Installation ==

1. Upload the entire `wirewheel-connector` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

== Configuration ==

1. You will find **Wirewheel Forms** menu in your WordPress admin screen.
2. Goto **Wirewheel Forms** > **Settings** and add the Wirewheel instance details.
3. List of configurations which are required are **WireWheel Instance ID**, **WireWheel Data path**, **Recaptcha Sitekey API link**, **WireWheel Data endpoint** and **Use Default Styling** checkbox
3. Goto Wirewheel Forms -> Forms page to add the forms.
